# CV Download Instructions

## How to Add Your CV PDF

1. **Place your CV file** in this directory: `src/assets/cv/`
2. **Name it exactly:** `liynadah_hela_cv.pdf`
3. **File format:** Must be a PDF file

## File Location
```
src/assets/cv/liynadah_hela_cv.pdf
```

## Download Functionality

The CV download buttons are located on:
- ✅ Home page (Hero section)
- ✅ About page
- ✅ Navigation bar
- ✅ Footer

All buttons will automatically download the PDF file when clicked.

## Testing

After placing your CV:
1. Start the development server: `npm start`
2. Navigate to any page with a "Download CV" button
3. Click the button - it should download the PDF file

## Notes

- The file must be named exactly: `liynadah_hela_cv.pdf` (case-sensitive)
- The file must be a valid PDF
- The download will work in both development and production builds
